Zbox distro of OpenELEC (tested for 64-bit ION)

 - MythTV 0.26.0 support
   - LIRC support matches OpenELEC XBMC mappings
 - Joint installation of both XBMC and MythTV alongside
   each other
 - Toggling between apps mapped to HOME key on remote
 - Customized ratpoison install to support background
   images and hiding the cursor
 - Install system partition size doubled to support
   the extra applications
 - Additional utilities installed by default such as
   rsync, iperf, etc.
 - Shell on tty2 always started even if non-debug
   version is running
